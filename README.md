# BeautyOfFractals iOS

## iOS version of Beauty of Fractals

### Version 1.3 improves deep-zoom rendering with adaptive preview and high-precision refinement, and adds Undo navigation and introduces the Watch Companion.
### Version 1.4 introduces the Precision / Refinement fix.

Beauty of Fractals is an interactive Mandelbrot and fractal explorer using Metal rendering, touch controls, color palettes, high precision preview, and PNG export.
This repository contains the iOS-specific source files and assets.

The shared project and privacy policy are available here:
https://github.com/LutzRFrank/BeautyOfFractals
